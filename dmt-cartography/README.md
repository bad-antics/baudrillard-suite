## ◈ RESEARCH AREAS

### ▸ HYPERSPACE TOPOLOGY

Mapping the geometry of altered states:

```python
from dmt_cartography import HyperspaceMapper

mapper = HyperspaceMapper()

# Load experience report dataset
reports = mapper.load_reports(
    sources=["erowid", "dmt_nexus", "academic"],
    min_reports=10000
)

# Extract spatial descriptors
topology = mapper.analyze_topology(reports)

print(f"▸ HYPERSPACE TOPOLOGY")
print(f"  Dimensions reported: {topology.avg_dimensions}")
print(f"  Non-Euclidean geometry: {topology.non_euclidean_pct}%")
print(f"  Infinite recursion: {topology.recursion_pct}%")
print(f"  Impossible objects: {topology.impossible_pct}%")

# Common spatial features
for feature in topology.common_features[:10]:
    print(f"  - {feature.name}: {feature.frequency}%")
```

### ▸ ENTITY CLASSIFICATION

Cataloging the beings encountered:

```python
from dmt_cartography import EntityClassifier

classifier = EntityClassifier()

# Analyze entity reports
entities = classifier.analyze(reports)

# Major entity categories
print(f"▸ ENTITY TAXONOMY")
for category in entities.categories:
    print(f"\n  {category.name}")
    print(f"    Frequency: {category.frequency}%")
    print(f"    Communication: {category.communication_type}")
    print(f"    Attitude: {category.predominant_attitude}")
    print(f"    Consistency: {category.cross_cultural_consistency}%")

# The infamous "machine elves"
elves = entities.get_type("machine_elves")
print(f"\n  Machine Elves (Tykes)")
print(f"    Reports: {elves.report_count}")
print(f"    Activities: {', '.join(elves.common_activities)}")
print(f"    Teaching: {elves.teaching_frequency}%")
```

### ▸ PATTERN CONSISTENCY ANALYSIS

Cross-cultural, cross-individual pattern matching:

```python
from dmt_cartography import ConsistencyAnalyzer

consistency = ConsistencyAnalyzer()

# Compare reports across cultures
cultural = consistency.cross_cultural(
    cultures=["western", "amazonian", "east_asian"],
    reports_per_culture=1000
)

print(f"▸ CROSS-CULTURAL CONSISTENCY")
print(f"  Entity types match: {cultural.entity_match}%")
print(f"  Space descriptions match: {cultural.space_match}%")
print(f"  Message themes match: {cultural.message_match}%")
print(f"  Visual motifs match: {cultural.visual_match}%")

# If random, expect ~0%. If >50%, something interesting...
print(f"\n  Random baseline: 5-10%")
print(f"  Actual: {cultural.overall_match}%")
print(f"  Statistical significance: p < {cultural.p_value}")
```
